# Dots
Este repositorio contiene mi configuracion de Hyprland.
### Instalación
Solo debes ejecutar el archivo hyprland.sh. En caso de fallar el script puedes copiar cada carpeta de forma manual.
Si lo prefieres, ejecuta el archivo hyprland_zenity.sh el cual permite una instalacíon de forma gráfica. Para ello debes tener instalado zenity, aunque por lo general ya viene instalado en la mayoria de las distribuciones linux.
## Hyprland
La configuracion incluye el gestor de ventanas hyprland pero esta misma configuracion puede ser instalada en distribuciones con el gestor de paquetes dnf, pacman y apt(en proceso).
